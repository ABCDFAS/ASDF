本机配置：

| CPU  | i5-8250u                                        |
| ---- | ----------------------------------------------- |
| 内存 | 4GB DDR4 2400                                   |

  - 声卡有时可能会耳机不太好 重启即可 估计死了
  - 网卡以太网可以的 无线我上了94360cs2 正常
  - 亮度已修复 可大幅度调
  - usb驱动已成功
  - 蓝牙正常
  - 触控板支持Apple多点式触控
  - 电池显示正常
  - 睡眠暂时可以 时间长自动唤醒 10.14.6以下可能会卡黑屏
